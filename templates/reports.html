Genera report sull'utilizzo degli estratti allergici

Report Utilizzo Annuale

Genera un report di tutti gli estratti utilizzati durante un anno specifico.

Informazioni Report

Il report annuale contiene le seguenti informazioni:

  • Nome estratto
  • Tipo (inalante, alimentare, controllo)
  • Numero lotto
  • Produttore
  • Data inizio utilizzo
  • Data fine utilizzo
  • Nome pannello
I report sono generati in formato CSV che può essere aperto in Excel o altre applicazioni di fogli di calcolo.

Funzionalità Future per i Report

Nelle versioni future, saranno disponibili le seguenti funzionalità di reporting:

Avvisi Scadenza

Report sugli estratti in prossimità della data di scadenza.

Tendenze di Utilizzo

Report sulle tendenze di utilizzo degli estratti nel tempo.

Distribuzione Estratti

Report sulla distribuzione dei tipi di estratto.
