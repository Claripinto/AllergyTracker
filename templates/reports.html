Reports

Generate reports of allergy extract usage

Annual Usage Report

Generate a report of all extracts used during a specific year.

Report Information

The annual report contains the following information:

  • Extract name
  • Type (inalante, alimentare, controllo)
  • Lot number
  • Manufacturer
  • Start date of usage
  • End date of usage
  • Panel name
Reports are generated in CSV format that can be opened in Excel or other spreadsheet applications.

Future Report Features

In future versions, the following reporting features will be available:

Expiration Alerts

Reports on extracts approaching expiration date.

Usage Trends

Reports on extract usage trends over time.

Extract Distribution

Reports on extract type distribution.
